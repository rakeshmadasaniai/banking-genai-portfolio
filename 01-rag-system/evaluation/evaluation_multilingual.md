## How to run

```python
results = []

for item in multilingual_queries:
    result = run_query(item["query"], mode=item["mode"])
    results.append({
        "language": item["language"],
        "mode": item["mode"],
        "query": item["query"],
        "answer": result["answer"],
        "chunks_retrieved": result["retrieved_chunks"],
        "latency_ms": result["latency_ms"],
        "grounded": result["grounded"],
        "confidence": result["confidence"],
    })
```

For each query record: retrieved chunks (rank 1–5), whether the answer was grounded, completeness, latency, and for Auto mode which model was selected.
